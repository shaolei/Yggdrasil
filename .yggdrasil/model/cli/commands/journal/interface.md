# Journal Commands Interface

| Function | Command | Key options / behavior |
| -------- | ------- | ----------------------- |
| `registerJournalAddCommand` | journal-add | --note \<text\> (required), --target \<node-path\>. |
| `registerJournalReadCommand` | journal-read | No options. |
| `registerJournalArchiveCommand` | journal-archive | No options. |

Errors to stderr, process.exit(1) on failure.
